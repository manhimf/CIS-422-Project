/AppDelegate.swift  -- Configures the application.
/SceneDelegate.swift -- Loads window (.storyboard) on opening app and entering background.
/Main.storyboard -- Contains all User Inteface buttons, text, and spinners with constraints.
/ViewController.swift -- Contains all code being processed once the application loads and runs.
/LaunchScreen.storyboard -- Contains display (text) as application load and during authorization.
/Info.plist -- Contains pregenerated values, and values for background operation and location serices. Used by AppDelegate.swift
/logo.png -- Simple red pin location icon for the application
